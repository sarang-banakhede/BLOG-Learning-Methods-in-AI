A Comprehensive Guide to Learning Methods in AI

Supervision Based

Supervised Learning

  1. Traditional/strong supervised learning

    A type of machine learning where the model learns from labelled data, mapping input features to known output labels.

    Image classification, where a model is trained on a dataset of labelled images (e.g., "cat," "dog," "bird") and learns to classify new, unseen images.

  2. Transfer learning

    A technique where a model developed for one task is reused as the starting point for a model on a second, related task.

    Using a pre-trained image recognition model (e.g., trained on ImageNet) and fine-tuning it for a specific task like identifying different breeds of dogs.

  3. Weakly supervised learning

    Learning approaches that can train models with limited, imperfect, or inaccurate supervision.

    • Active Learning: An approach where the model actively selects the most informative data points for labelling by human experts.
      In text classification, the model might select sentences it's most uncertain about for human labelling, improving its performance efficiently.
    • Self-supervised learning: The model selects the elements with the highest confidence score and uses them to label the similar ones.
      In natural language processing, predicting the next word in a sentence based on the previous words, allowing the model to learn language patterns without explicit labelling.
    • Multiple instance learning: A form of weakly supervised learning where labels are provided for sets or bags of instances, rather than individual instances.
      In drug discovery, where a bag of molecular conformations is labelled as active or inactive, but the specific conformation responsible for the activity is unknown.

Unsupervised Learning

A type of machine learning where the model learns patterns from unlabelled data.

Clustering customer data to identify distinct market segments without predefined categories.

Reinforcement Learning

A learning paradigm where an agent learns to make decisions by taking actions in an environment to maximize a cumulative reward.

Training a game-playing AI to master chess or go by playing many games and learning from the outcomes.

Moment of Learning

Offline Learning

The model is trained from a fixed dataset that has been collected and labeled before deployment.

Training a spam detection model on a large corpus of labelled emails before deploying it in an email service.

Online Learning

  1. Continual / Incremental Learning

    The model continues to learn and adapt from a stream of data after initial training.

    A recommendation system that updates its model as users interact with content, continuously improving its suggestions.

  2. Federated / Collaborative Learning

    The model is trained across multiple decentralized edge devices or servers holding data locally without exchanging them.

    Multiple hospitals training a shared diagnostic model on their local patient data without exchanging the sensitive information.

Task Based

Single-Task Learning

The model is trained to perform a single specific task.

A sentiment analysis model trained solely to classify text as positive, negative, or neutral.

Multi-Task Learning

The model is trained to perform multiple related tasks simultaneously.

A natural language processing model that can perform part-of-speech tagging, named entity recognition, and sentiment analysis on the same input text.

Contrastive Learning

A technique where the model learns to distinguish between similar and dissimilar data points in a representation space.

Training a face recognition system by learning to maximize the similarity between different images of the same person while minimizing similarity with images of different people.

N-shot Learning

  1. Zero-shot Learning

    The ability of a model to recognize or classify instances of classes that were not seen during training.

    A language model answering questions about a topic it wasn't explicitly trained on, using its general knowledge and reasoning abilities.

  2. One-shot Learning

    The ability to learn a task from just one example.

    A facial recognition system that can identify a person after seeing only one photo of them.

  3. Few-shot Learning

    The ability to learn a task from a small number of examples.

    A machine translation model that can adapt to a new language pair after seeing only a few dozen translated sentences.

Adversarial Learning

A technique involving the training of two competing models: one that generates adversarial examples, and another that learns to resist or detect these attacks.

Training an image classification model to correctly classify images even when they've been subtly altered to fool the model.

Meta Learning

An approach where the model learns how to learn, improving its ability to acquire new skills or adapt to new tasks quickly.

A model that learns general strategies for image classification tasks, allowing it to quickly adapt to classifying new types of objects with minimal additional training.
